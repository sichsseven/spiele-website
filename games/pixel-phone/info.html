Über das Spiel

Pixel Phone ist eine Stille Post zum Mitspielen: Es geht abwechselnd ums Schreiben und Zeichnen. Der Host legt einen Raum mit sechsstelliger Zahl an, Freunde treten mit dem Code bei. Pro Runde siehst du immer den Beitrag eines anderen Spielers auf einem „Heft“ — dein eigener Start kommt dir erst in der Auflösung wieder entgegen. So viele Runden wie Mitspieler sind spielen alle parallel mit; am Ende wird jede Kette Schritt für Schritt vorgelesen und gezeigt. Du brauchst nur dein PIXELZONE-Konto; Namen kommen automatisch aus dem Login. Auf PC, Tablet und Handy dieselbe Leinwand — am Tablet am besten mit dem Finger zeichnen. Nach der Auflösung bleibt ihr im selben Raum; der Host kann direkt eine neue Runde starten.
